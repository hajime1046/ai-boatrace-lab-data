# AIボートレース研究所 — 予想・成績データ

「AIボートレース研究所」（愛称: 艇研）が毎日公開している競艇（ボートレース）
AI予想モデルの**全記録**です。

## このリポジトリは何か

AIボートレース研究所は、機械学習モデルによる競艇の予想を無料で公開し、
その**的中・不的中を含めた全成績を機械集計で検証可能にする**ことをコンセプトにした
検証メディアです。このリポジトリはその技術的な裏付けとして、

- 毎朝生成される予想（`predictions/`）
- 毎晩の答え合わせ・回収率などの集計（`grades/`）

を**そのまま日次でコミット**しています。都合の良い日だけ公開する・成績の良い予想だけ
載せるといった恣意的な選別はできない構造になっています（コミット履歴そのものが
「いつ・何を予想したか」の証跡です）。

## 重要な原則

- **予想は `generated_at` の時点で確定し、対象レースの締切前に公開されます。**
  結果を見てから予想を書き換えることはありません（そもそも締切前にコミットされるため、
  技術的に不可能です）
- 外れた予想も削除・修正せず、そのまま記録に残ります
- 成績（的中率・回収率）は `grades/summary.json` で**全期間を毎回ゼロから再集計**しており、
  都合の良い期間だけを切り出す・特定のレースを除外するといった操作はしていません
- 単勝（1着的中）の100円均一買いを基準にした機械集計です。実際の払戻データ
  （的中時のみ判明する構造）を使っているため、この集計方法自体に恣意性はありません

このリポジトリはデータの一次配布のみを目的としています。予想の解説・根拠・記事は
本体サイト／X（旧Twitter）／YouTubeで発信しています。

## データスキーマ

### `predictions/YYYY-MM-DD.json` — 朝に公開する予想

```json
{
  "schema_version": 1,
  "date": "2026-07-13",
  "generated_at": "2026-07-13T09:00:03+09:00",
  "model_id": "morning_logit_lane_v1_20260713T215525",
  "races": [
    {
      "venue": { "code": "03", "name": "江戸川" },
      "race_no": 1,
      "close_scheduled_at": "2026-07-13T11:14:00+09:00",
      "probs": { "1": 0.582, "2": 0.2295, "3": 0.0991, "4": 0.0553, "5": 0.0127, "6": 0.0215 },
      "top_pick": 1,
      "confidence": 0.582
    }
  ],
  "summary": { "scheduled": 156, "predicted": 156, "failed": 0, "failed_races": [] }
}
```

| フィールド | 意味 |
|---|---|
| `generated_at` | 予想の生成時刻（JST）。**この時刻より後にレース結果を反映することはない** |
| `model_id` | 使用した予想モデルのバージョン識別子 |
| `probs` | 枠番(1〜6)ごとの勝率予測 |
| `top_pick` | モデルの本命（最も勝率が高いと予測した枠番） |
| `confidence` | `top_pick`の予測勝率 |

### `grades/YYYY-MM-DD.json` — 夜の答え合わせ

```json
{
  "schema_version": 1,
  "date": "2026-07-13",
  "generated_at": "2026-07-13T23:30:12+09:00",
  "model_id": "morning_logit_lane_v1_20260713T215525",
  "races": [
    {
      "venue": { "code": "03", "name": "江戸川" },
      "race_no": 1,
      "top_pick": 1,
      "confidence": 0.582,
      "status": "confirmed",
      "actual_winner": 1,
      "hit": true,
      "stake": 100,
      "payout_yen": 150
    }
  ],
  "counts": {
    "scheduled": 156, "confirmed": 144, "pending": 12,
    "hits": 85, "hit_rate": 0.5903,
    "stake": 14400, "return": 13600, "roi": 0.9444
  },
  "by_confidence_bucket": { "0.8": { "n": 12, "hits": 9, "roi": 0.95 } }
}
```

- `status: "pending"` のレースは、答え合わせ実行時点でまだレースが終わっていない
  （主にミッドナイト最終レース）ことを示します。翌日以降の実行で自動的に確定します
- `hit` / `payout_yen` は単勝100円買いを基準にした機械集計です

### `grades/summary.json` — 全期間の累計成績

```json
{
  "schema_version": 1,
  "generated_at": "2026-07-13T23:30:12+09:00",
  "n_days": 1,
  "overall": { "n_confirmed": 144, "n_hits": 85, "hit_rate": 0.5903, "roi": 0.9444 },
  "confidence_0_8_plus_strategy": { "n_bets": 13, "hits": 10, "roi": 0.9538 },
  "daily_series": [ { "date": "2026-07-13", "confirmed": 144, "hits": 85, "roi": 0.9444 } ]
}
```

`grades/*.json`（日次ファイル）を毎回すべて読み直して再計算したものです。
特定の日だけ除外する・積み上げ集計にする、といった操作の余地がない設計です。

## 公開しないデータ

以下は競争優位・非公開方針のため、このリポジトリには含まれません。

- オッズデータ（EV戦略の検討に使用。非公開）
- 予想モデルの重み・パラメータそのもの

## 更新頻度

- 毎日 09:00（JST）: 当日の予想を追加
- 毎日 23:30（JST）: 前日までの答え合わせ・累計成績を更新

## ライセンス・利用について

このデータは検証・研究目的での参照を想定しています。転載・二次利用の際は
出典（AIボートレース研究所）を明記してください。

本データは投資判断を保証するものではありません。舟券の購入は自己責任で
行ってください。
